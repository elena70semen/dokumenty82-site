## Purpose

This is the Stage 18A source route-to-query-cluster map for `Документы для бизнеса`.

It maps approved routes to safe semantic clusters for Yandex-oriented SEO, implementation QA, internal linking and owner/legal review. It does not add public routes, does not use numeric frequency, does not make SERP claims and does not approve public launch.

## Source Basis

The map is controlled by the route registry, SEO canon, semantic core, Stage 16 semantic service map, service catalog, service passports, content/UX/front-end standards and live-launch gates in the source repository.

## Field Legend

Each route row includes:

- URL;
- route class;
- parent route;
- route group;
- service cluster;
- primary intent;
- secondary support intent;
- safe user wording;
- safe query variants;
- problem hooks;
- content angle;
- FAQ angle;
- related routes;
- routes to avoid cannibalizing;
- metadata direction;
- H1 direction;
- schema boundary;
- HOLD risks;
- implementation status.

Safe query variants are qualitative wording patterns only. They are not frequency data and must not be treated as ranking or demand claims.
